# טופס לקוחות חכם - לידרס

טופס אינטרנטי מודרני למילוי בריף ללקוחות לידרס, עם 6 מסכים נוחים, שמירה אוטומטית ושליחה לוובהוק של Make.

## התקנה

```bash
npm install
```

## הרצה במצב פיתוח

```bash
npm run dev
```

האתר יפתח בכתובת: http://localhost:3000

## בניית הפרויקט לפרודקשן

```bash
npm run build
npm start
```

## תכונות עיקריות

- **6 מסכים נוחים** - השאלון מחולק ל-6 מסכים עם 3-4 שאלות בכל מסך
- **שמירה אוטומטית** - הנתונים נשמרים אוטומטית ב-localStorage במהלך המילוי
- **וולידציה חכמה** - בדיקת שדות חובה ופורמטים נכונים
- **עיצוב מודרני** - עיצוב מקצועי ונקי עם תמיכה מלאה ב-RTL
- **שליחה לוובהוק** - כל הנתונים נשלחים אוטומטית ל-Make

## טכנולוגיות

- Next.js 14
- TypeScript
- Tailwind CSS
- React Hook Form
- Zod
- Axios

## מבנה הפרויקט

```
├── app/                    # Next.js App Router
│   ├── layout.tsx         # Layout ראשי
│   ├── page.tsx           # עמוד הבית
│   └── globals.css        # סגנונות גלובליים
├── components/            # קומפוננטות React
│   ├── StepperForm.tsx   # הטופס הראשי
│   ├── StepIndicator.tsx # אינדיקטור התקדמות
│   ├── FormField.tsx     # שדה טופס
│   └── NavigationButtons.tsx # כפתורי ניווט
├── lib/                   # פונקציות עזר
│   ├── formSteps.ts      # הגדרת המסכים
│   └── localStorage.ts   # ניהול localStorage
├── types/                 # הגדרות TypeScript
│   └── form.ts           # טיפוסים וסכמות
└── public/               # קבצים סטטיים
    └── logo.png          # לוגו לידרס
```

## מסכי הטופס

1. **פרטים כלליים** - תאריך, מגיש הבריף, שם הלקוח, המוצר/השירות, דד ליין
2. **רקע** - שוק וקטגוריה, תיאור המוצר, מתחרים, האתגר
3. **קהל היעד** - קהל היעד, מאפיינים, תובנות
4. **מטרות** - מטרות המהלך, תגובה רצויה, טיימינג
5. **תובנה ומסר** - תובנה, פתרון, מסר עיקרי, key takeaway
6. **דרישות ותקציב** - דרישות, סוג המהלך, תקציב, הערות
